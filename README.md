# AWS S3 Portfolio Website

A responsive personal portfolio website hosted using Amazon S3 Static Website Hosting.

## Project Overview

This project demonstrates how to deploy a static website on AWS using Amazon S3.

## AWS Services Used

- Amazon S3
- Static Website Hosting
- AWS IAM
- Bucket Policy

## Features

- Responsive Portfolio Website
- Resume Download
- Project Showcase
- Contact Information
- Static Website Hosting

## Project Architecture

Local Development
        ↓
GitHub Repository
        ↓
Amazon S3 Bucket
        ↓
Static Website Hosting

## Skills Demonstrated

- AWS Cloud
- Amazon S3

- Static Website Hosting
- Git
